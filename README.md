# 🔍 FindX_UTF8 進階內容搜尋工具

FindX_UTF8 是一個專為 Windows 環境打造的輕量級、帶有圖形介面 (GUI) 的本機檔案內容搜尋工具。
特別針對開發人員在搜尋 **Oracle 二進位檔案 (.fmb, .rdf)** 以及 **UTF-8 中文編碼** 時常遇到的亂碼與排版問題進行了深度優化。

## ✨ 核心特色

* **支援多語系與二進位檔：** 完美解析 UTF-8 中文，並內建「幽靈字元 (Null Byte)」淨化機制，讀取 `.fmb`、`.rdf` 等二進位檔時不會報錯崩潰，且能精準計算行數。
* **多關鍵字分組顯示：** 支援同時輸入多個關鍵字，搜尋結果會依據關鍵字自動分組排版，一目了然。
* **極速精簡模式：** 提供「僅顯示檔名」選項。勾選後只要在檔案內命中關鍵字便會立即跳出換下一個檔案，大幅提升大範圍搜尋的效能，也方便使用記事本查看結果。
* **一鍵匯出防呆：** 搜尋結果可一鍵匯出為 `.txt` 檔，且程式會自動寫入 `UTF-8 BOM` 標記，確保用任何編輯器開啟都不會產生亂碼。
* **安全中斷機制：** 內建「⏹️ 停止搜尋」按鈕，不小心搜錯大範圍資料夾時隨時可以優雅煞車，視窗不會卡死。

## 🚀 如何使用 (一般使用者)

1. 前往右側的 **[Releases]** 頁面。
2. 下載最新版的 `FindX_UTF8.exe`。
3. 免安裝，直接雙擊執行檔即可開啟。
4. 輸入你要找的關鍵字、選擇目標資料夾、設定檔案類型 (如 `*.fmb, *.sql`)，點擊「開始搜尋」即可。

## 💻 給開發者 (如何從原始碼執行與打包)

本專案使用 Python 與內建的 `tkinter` 模組開發，無需安裝龐大的第三方 GUI 套件。

**1. 執行原始碼:**
```bash
python FindX_GUI.py
```

**2. 打包成單一 EXE 執行檔:**
請先安裝 PyInstaller：
```bash
pip install pyinstaller
```
接著執行打包指令：
```bash
pyinstaller --onefile --noconsole FindX_GUI.py
```
打包完成後，可在 `dist` 資料夾中找到生成的 `.exe` 檔案。
